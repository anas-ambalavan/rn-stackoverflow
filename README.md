## Expo Router Example

Use [`expo-router`](https://expo.github.io/router) to build native navigation using files in the `app/` directory.

## Quick Start
``` bash
  # Install dependencies
  npm install
  
  # Run commands
  expo start
  Press a: open Android
  Press i: open iOS simulator
```
